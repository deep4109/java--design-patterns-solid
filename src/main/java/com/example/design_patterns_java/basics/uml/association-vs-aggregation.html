There are four kinds of Class relationships Association: uses a Ex:a Class Man uses a Class Pen ( Pen is still there when man die ) Aggregation: has a Ex:a Class Man has a Class Car ( Car is still there when Man die ) Composition: owns a Ex:a Class Man owns a Class Heart ( When Man die, Heart die ) Inheritance: is a Ex:a Class Man is a Class Human ( Man is a Human ) A relationship between classes of objects https://stackoverflow.com/questions/4298177/association-vs-aggregation
